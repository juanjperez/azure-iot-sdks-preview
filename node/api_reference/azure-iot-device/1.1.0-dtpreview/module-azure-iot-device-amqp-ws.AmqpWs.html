Class: AmqpWs

azure-iot-device-amqp-ws. AmqpWs

Constructs an AmqpWs object that can be used on a device to send and receive messages to and from an IoT Hub instance, using the AMQP protocol over secure websockets. This class overloads the constructor of the base module:azure-iot-device-amqp:Amqp class from the AMQP transport, and inherits all methods from it.
