Class: Client

azure-iot-device. Client

Creates an IoT Hub device client. Normally, consumers will call the factory method, fromConnectionString, to create an IoT Hub device client.

Constructor

new Client(transport, connStr, blobUploadClient)

Parameters:
Name Type Description
transport Object

An object that implements the interface expected of a transport object, e.g., Http.
connStr String

A connection string (optional: when not provided, updateSharedAccessSignature must be called to set the SharedAccessSignature token directly).
blobUploadClient Object

An object that is capable of uploading a stream to a blob.
